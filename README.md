## Pre-reqs

- Github Account
- Github Repo with 1 or more manifest files

## Exercise

For this exercise you will need to set up a Github Repo with a Github Action that will the Socket CLI. https://docs.socket.dev/docs/socket-for-github-actions

## End Result Requirements

1. Github Action is configured to Run Socket
2. A Github PR with a comment from the Github Action (Dependency Overview or Security Issue or both)
